# Projet de Gestion TPGIT

Ce projet est un laboratoire d'expérimentation pour maîtriser les commandes Git avancées (Stash, Cherry-pick, Rebase).

## Fonctionnalités principales
* Gestion des sauvegardes temporaires avec la commande stash
* Extraction de commits ciblés via cherry-pick
* Alignement et lissage d'historique avec rebase
